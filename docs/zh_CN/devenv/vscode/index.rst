=======
VS Code
=======

`VS Code`_ 是微软出品的一个免费 `IDE` ，支持多种编程语言，界面优雅。

快速上手
========

先到 `VS Code`_ 官网，根据你的操作系统下载下载对比版本。以 `OSX` 为例：

.. figure:: /_images/devenv/vscode/index/5ffff373cf9a218e0477e87e8397e73d.png
    :width: 580px

下载完毕后，解开 `ZIP` 压缩包，然后将程序拖进应用程序目录即可：

.. figure:: /_images/devenv/vscode/index/85cbeae211b8abffdab67befa295bb69.png
    :width: 385px
