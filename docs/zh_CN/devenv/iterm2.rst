======
iTerm2
======

程序开发离不开一个好用的终端程序。

然而， `OSX` 系统内置的终端程序( `Terminal.app` )不仅长得丑，还非常难用：

.. figure:: /_images/devenv/iterm2/b17bf847ac17724ba7729ddbec58973c.png
    :width: 235px

    丑陋的内置终端

好在，我们可以安装另一个终端应用—— `iTerm2 <https://www.iterm2.com/>`_ ：

.. figure:: /_images/devenv/iterm2/730eb902850c1d5d175580597d7f034f.png
    :width: 235px

    丑陋的内置终端

`iTerm2` 不仅颜值高，功能还十分强大！

安装
====

点击 `iTerm2 下载 <https://www.iterm2.com/downloads.html>`_ ，选择稳定版( `Stable` )并下载。
下载完成后，将程序拖拽到应用程序里即可。
