======
初始化
======

新 ``Mac Book`` 拿到手后，系统空荡荡的，说不上好用。
要让 ``Mac Book`` 发挥出强大能力，需先进行一些设置。

系统偏好设置
============

系统偏好设置相当于 `Windows` 下的控制面板，系统相关设置都在这操作。
接下来，先来看看如何调出系统偏好设置。

第一步，点击屏幕左上角的苹果图标，弹出菜单：

.. figure:: /_images/initialization/4af680e4ad6675e0243215dfc3c73b56.png
    :width: 216px

第二步，点击系统偏好设置，弹出面板：

.. figure:: /_images/initialization/4d7bdfc889721c05edaf5308e34b9359.png
    :width: 334px

程序坞
======

程序坞是 `OSX` 下的任务条，默认显示在屏幕下方：

.. figure:: /_images/initialization/1414e7638d9ef695bf3357f8e39ccd0b.png
    :width: 720px

屏幕本来就是长方形的，高度不高，程序坞又占用了不少空间。
因此，可以将程序坞调到屏幕左边。
打开系统偏好设置，点击程序坞，选择左边：

.. figure:: /_images/initialization/08345a89867185b71a0f5c242d1048c7.png
    :width: 334px

调整后，看起来科学了许多：

.. figure:: /_images/initialization/6135244d97a523b88d905a139020b034.png
    :width: 720px

输入法
======

系统内置的输入法非常难用，建议换成 `搜狗 <https://pinyin.sogou.com>`_ 。

搜狗输入法安装完毕后，可以将一些然并卵的输入法给删掉，只留搜狗和英文：

.. figure:: /_images/initialization/22e7e75396c980716cbc0e706bef0f79.png
    :width: 445px

    系统偏好设置 >> 键盘 >> 输入法

安装应用
========

App Store
---------

应用获取的首选通道是 `App Store` ，操作非常简单。

第一步，打开 `App Store` ，在搜索框输入应用名并按下回车开始搜索：

.. figure:: /_images/initialization/3b729de3a96af9fe81eecfebba9db55a.png
    :width: 720px

第二步，在结果中找到想要的应用，先点击获取，再点击安装：

.. figure:: /_images/initialization/6e7dc8a700340d2611740505e6c1a983.png
    :width: 720px

安装完成后，点击打开即可启动应用。

另外，点击启动台可以看到系统所有应用，点击应用便可以启动。
